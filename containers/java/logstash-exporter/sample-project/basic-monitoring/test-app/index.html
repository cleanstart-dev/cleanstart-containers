About This App

This test application is designed to:

  • Generate sample log data for testing Logstash pipelines
  • Demonstrate different log levels (info, warning, error)
  • Provide a simple interface for log generation
  • Show integration with Logstash monitoring

How It Works

  1. Click any of the log generation buttons above
  2. The app generates a log entry with timestamp and metadata
  3. Logs are sent to Logstash via TCP connection
  4. Logstash processes and filters the logs
  5. logstash-exporter collects metrics from Logstash
  6. Monitor the metrics at the exporter endpoint
